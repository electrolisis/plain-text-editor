Basic plain text editor (Qt)

Folhium project

https://github.com/electrolisis/plain-text-editor

--------------------------------------------------------------------------------

## Dependencies

Qt Toolkit

debian/*buntu:

qt5:
    # apt install qtbase5-dev

qt6:
    # apt install qt6-base-dev
    
## Build / install
    
    $ mkdir build
    $ cd build
    $ cmake ..
    $ make
    # sudo make install
